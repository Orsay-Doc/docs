**Two tabs:**

- **Conversational** — List of active AI agents. Each can be edited, configured, or deleted. Agents handle and respond to leads.
- **Filtering** (coming soon) — Agents for classifying leads before contacting them.

Sub-pages: Create agent, Edit agent (for both conversational and filtering).
