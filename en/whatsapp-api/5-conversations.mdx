Every lead that engages with your WhatsApp Business sequence appears in **Conversations**.

## Follow conversations in Orsay

The **Conversations** page lists every active thread, classified by the AI:

- **Positive** — booked or about to book
- **Engaged** — talking but not qualified yet
- **Negative** — not interested
- **Undetermined** — no signal yet

Filter by classification, sort, search, and click any thread to read the full message history. Re-classify manually if the AI is wrong — corrections feed [Conversation Improvements](/en/overview/conversation-improvements) so the agent learns.

## Take over a conversation

For high-value leads, step in yourself:

- Open the thread in Orsay and **type a reply** directly.
- Send a voice note from the input bar, or use **Text-to-Audio** with your cloned voice. → [Voice cloning](/en/guides&tutorials/voice-cloning)

When you reply, the AI agent **disables itself for that thread automatically**. Re-enable it with the toggle when you're done.

<Note>
Unlike Personal WhatsApp, the WhatsApp Business API does **not** sync messages to a phone. All takeover happens inside Orsay.
</Note>

## Watch the 24-hour window

Each thread shows time remaining in the 24-hour reply window. After it expires, the agent can no longer free-text — only an approved template can re-open the conversation.
