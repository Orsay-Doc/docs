## What Orsay AI Does on Instagram (Inbound)

Orsay AI captures incoming leads with instant DM responses and automated follow-ups that convert prospects into booked appointments.

**Instagram Inbound Features:**

- **DM Automation:** Sub-5-minute responses with intelligent follow-up sequences
- **Comment-to-DM Conversion:** Auto-engages users who comment on your content
- **Story Reply Handling:** Automatically responds to Instagram story replies

## Inbound triggers (no limitations)

- **A Message received**
- **A Story replied**
- **A comment on one of your posts**

## Keyword & Post Targeting (Story Reply & Comment)

For **Story Reply** and **Comment** triggers, you can refine when a sequence activates using **keywords** and **post/story targeting**:

| Sequence Type | Keywords | Target Posts/Stories | Behavior |
|---|---|---|---|
| **Global** | None | None | Triggers on all comments/replies (fallback) |
| **Keyword-specific** | ✅ Set | None | Triggers only when text contains a keyword |
| **Post/Story-scoped** | Optional | ✅ Set | Triggers only for selected posts/stories |

<Info>
**Matching priority:** When a comment or story reply arrives, sequences are evaluated in this order:
1. **Post/Story-scoped sequences** — checked first. If a match is found and the sequence has keywords, the text must also contain a keyword.
2. **Keyword-specific sequences** — checked next. First keyword match wins.
3. **Global sequence** — used as fallback if no other sequence matched.
</Info>

<Tip>
**Use case example:** Create a keyword sequence with "price" and "pricing" to route pricing inquiries to a specialized agent, while a global sequence handles all other comments with a general-purpose agent.
</Tip>

**📝 Story reply (your account)**
When a reply is made to an Instagram story, this action is triggered

<Expandable title="Step">
    - Click on ➕ (+)
    - Add action step
    - **Action step 1**

    <ResponseField name="Account :">
    - **Select your account :** Select the account for Instagram.
    </ResponseField>

    <ResponseField name="Keywords (optional) :">
    - **Add trigger keywords** to make this a **specific sequence**. It will only trigger when the story reply contains one of those keywords.
    - If no keywords are added, this is a **global sequence** that triggers on all story replies.
    - Keywords are **case-insensitive** and use **OR logic** (any keyword match triggers the sequence).
    </ResponseField>

    <ResponseField name="Target Stories (optional) :">
    - **Select specific active stories** to limit the sequence to replies on those stories only.
