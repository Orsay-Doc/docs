# docs

## Run locally

1. **Install the Mintlify CLI** (requires Node.js 19+):

   ```bash
   npm i -g mintlify
   ```

2. **Start the dev server** from this directory:

   ```npx mint dev
   ```

   The site will be available at **http://localhost:3000**.

**Optional:** Use a different port with `mintlify dev --port 3333`, or run without a global install: `npx mintlify dev`.
