**Le menu se trouve en haut à gauche.**

Ce menu vous permet de gérer vos conversations, la qualification des leads, l’envoi de messages et de fichiers.

Pour consulter vos conversations, vous devez d’abord créer votre [séquence](/fr/guides&tutorials/sequences).

## Si la séquence est correctement créée

La page s’affichera comme ceci :

<img style={{ borderRadius: "0.5rem" }} src="/image_conversations/1.png" />

Si vous souhaitez trier vos conversations :

- Cliquez sur **“plus récentes”**
- Ici, vous pouvez choisir de trier entre **“plus récentes”** et **“plus anciennes”**
  <img style={{ borderRadius: "0.5rem" }} src="/image_conversations/2.png" />

Si vous souhaitez trier vos conversations par type de lead :

Cliquez sur **“afficher :”**

Vous pouvez choisir entre :

- **positif**
- **engagé**
- **indéterminé**
- **négatif**

<img style={{ borderRadius: "0.5rem" }} src="/image_conversations/3.png" />

Vous pouvez également reclasser les conversations manuellement. Pour cela :

- Cliquez sur la flèche (engagé, positif, etc.)
- Sélectionnez l’état de la conversation
- Cliquez sur **“Confirmer”**

La conversation sera directement placée dans la section appropriée.

<img style={{ borderRadius: "0.5rem" }} src="/image_conversations/4.png" />
<img style={{ borderRadius: "0.5rem" }} src="/image_conversations/5.png" />
<img style={{ borderRadius: "0.5rem" }} src="/image_conversations/6.png" />
