Sur la page **Agents**, vous pouvez visualiser, créer et gérer vos agents IA. Chaque agent est conçu pour automatiser l’engagement des leads et gérer les conversations pour votre espace de travail.

- **Cliquez sur Agents**

<img style={{ borderRadius: "0.5rem" }} src="/image_agents/1.png" />

Lorsque vous accédez pour la première fois à la page **Agents**, vous pouvez voir :

- Une liste des agents existants (si certains ont déjà été créés)
- Les noms des agents, leurs statuts (Actif/Inactif) et l’**objectif principal** de chaque agent
  <img style={{ borderRadius: "0.5rem" }} src="/image_agents/2.png" />

## Créer un agent

**Lorsque vous commencez à créer votre agent IA sur Orsay.ai, vous verrez deux options. Pensez-y comme deux chemins différents :**

### 1 - Agent à prompt personnalisé :

L’**Agent à prompt personnalisé** vous permet d’écrire des instructions détaillées pour façonner le comportement de l’agent. Cela offre plus de contrôle, mais est mieux adapté aux utilisateurs avancés.

### 2 - Création guidée d’agent IA :

La **création guidée** est rapide et facile pour les débutants. Vous répondez à quelques questions et votre agent est prêt. Pour la plupart des utilisateurs, cette option guidée est la plus simple et recommandée.

1. **Cliquez sur “GUIDED AI AGENT SETUP”**
2. **Remplissez les informations de l’agent** :
   - **Nom de l’agent** : Donnez un nom reconnaissable à votre agent (ex. : “Rico Coolz”)
   - **Personnalité de l’agent** : Choisissez une personnalité prédéfinie pour votre agent. Exemple : “Homme 25 ans – Confiant, Ingénieux, Déterminé, Charismatique”
   - **Langue de l’agent** : Choisissez la langue principale des réponses de l’agent
   - **Description de l’entreprise (1 phrase)** : Fournissez une brève description de votre entreprise pour que l’agent puisse s’y référer lors des interactions avec les leads
   - **Votre offre commerciale** : Décrivez ce que l’IA doit promouvoir ou mettre en avant (ex. : produit, service, promotion spéciale) et partagez tout ce que l’IA doit savoir sur votre offre
   - **Questions de qualification** : Listez les questions que votre agent doit poser pour qualifier les leads (ex. : situation actuelle, budget, intérêt). Cliquez sur + pour ajouter plusieurs questions
   - **Stratégie d’appel** : Indiquez si l’agent doit vérifier si le lead accepte un rappel ou l’inviter à réserver un appel directement (**lien de réservation requis**). Il s’agit de choisir entre un suivi passif ou une prise de rendez-vous active
   - **Fréquence des emojis** : Contrôlez la fréquence d’utilisation des emojis par l’agent (nous recommandons 25 %)
   - **Liste d’emojis (optionnel)** : Vous pouvez choisir les emojis que l’agent utilisera dans ses réponses
   - **Ton direct** : Déterminez si le ton de l’agent doit être plus empathique ou plus direct (**0 = empathique, 10 = très direct pour obtenir plus d’appels**, nous recommandons 2)
3. **Cliquez sur “Save draft”** puis **“Publish agent” (en haut à droite)** pour finaliser et ajouter l’agent à votre liste
   <img style={{ borderRadius: "0.5rem" }} src="/image_agents/3.png" />

### Modifier un agent

Vous pouvez modifier les informations d’un agent à tout moment pour mieux répondre à vos besoins :

1. **Localisez l’agent** dans la liste sur la page **Agents**
2. **Cliquez sur les trois points (ellipsis)** à côté du nom de l’agent
3. **Sélectionnez “Edit”** : Vous accédez au même formulaire que lors de la création de l’agent
   <img style={{ borderRadius: "0.5rem" }} src="/image_agents/4.png" />{" "}
4. **Apportez vos modifications** : Ajustez n’importe quel champ, comme le nom, la personnalité, la langue ou la stratégie de l’agent
5. **Cliquez sur “Publish agent”** : La configuration de l’agent sera mise à jour
   <img style={{ borderRadius: "0.5rem" }} src="/image_agents/5.png" />

**Lors de la modification, vous pouvez également prévisualiser l’agent pour vérifier le déroulement de la conversation (Test de conversation).**
