title: "📣 Campagnes"

---

Une **Campagne** est un envoi en masse one-shot à des leads WhatsApp existants. Contrairement à une séquence (qui tourne en continu sur un déclencheur), une campagne part **une seule fois**, vers l'audience que vous sélectionnez, avec le message que vous écrivez.

<Note>
**WhatsApp uniquement.** Les campagnes fonctionnent sur WhatsApp Personnel et l'API WhatsApp Business. C'est l'outil parfait pour broadcaster quelque chose *maintenant* (un lancement, un événement, une promo) — et particulièrement puissant pour envoyer un **message vocal en premier contact**.
</Note>

## Quand utiliser une Campagne vs une Séquence

| Utilisez une **Séquence** quand… | Utilisez une **Campagne** quand… |
|---|---|
| Vous voulez une automatisation continue, basée sur déclencheur (chaque nouveau lead entre dans le flow) | Vous voulez un envoi unique vers une liste définie |
| Chaque lead est traité individuellement, à son rythme | Tous les leads reçoivent le message en même temps |
| Le premier message est un template qui tourne en permanence | Vous annoncez quelque chose *maintenant* |

## Envoyer une Campagne audio sur WhatsApp

Le use case phare : ouvrir une campagne avec un **message vocal** pour que les leads entendent votre vraie voix avant tout texte.

<Steps>
  <Step title="Ouvrez Campagnes">
    Dans la barre latérale, cliquez sur **[Campaigns](https://app.orsay.ai/campaigns)** → **+ Nouvelle campagne**.
  </Step>
  <Step title="Filtrez votre audience">
    Sur le panneau de gauche, affinez qui reçoit la campagne :
    - **Canal** — choisissez WhatsApp Personnel ou WhatsApp Business
    - **Classification** — Positif / Négatif / Engagé / Indéterminé
    - Recherchez par nom, téléphone, ou username

    Le panneau de droite se met à jour en direct avec les leads correspondants. Sélectionnez tout, ou choisissez manuellement avec les cases à cocher.
  </Step>
  <Step title="Composez le message">
    Dans l'éditeur de message :
    - Basculez le type de message sur **🎙️ Message vocal**.
    - **Enregistrez** directement dans l'éditeur ou **importez** un fichier audio (max 25 Mo ; MP3, M4A, WAV, OGG, AAC, WebM).
    - Ajoutez optionnellement un message texte de suivi envoyé juste après le vocal.
    - Personnalisez le texte avec `{{first_name}}`, `{{last_name}}`, `{{username}}`.

    L'aperçu en direct à droite montre le message exact que chaque lead recevra.
  </Step>
  <Step title="Envoyez ou planifiez">
    Cliquez sur **Envoyer maintenant** pour broadcaster tout de suite, ou **Planifier** pour fixer une date/heure future.
  </Step>
</Steps>

## Campagnes texte uniquement

Le vocal n'est pas obligatoire — vous pouvez envoyer une campagne texte exactement de la même façon (sautez l'étape vocal dans l'éditeur). Mêmes filtres, mêmes variables, même planification.

## Après la campagne

Chaque réponse atterrit dans [Conversations](/fr/personal-whatsapp/5-conversations) et est gérée par l'agent assigné (ou par vous si vous reprenez la main). Suivez taux d'ouverture, taux de réponse et taux positif dans [Analytics](/fr/overview/analytics).

## Import de leads en masse (CSV)

Si votre audience n'est pas encore dans Orsay, uploadez un CSV via **[Intégrations](https://app.orsay.ai/integrations) → [Bulk Leads](https://app.orsay.ai/integrations/bulk-leads) → Upload Files**. Puis lancez une Campagne sur les leads fraîchement importés.

→ Pour une automatisation continue sur chaque nouveau lead, utilisez plutôt une [séquence WhatsApp](/fr/personal-whatsapp/3-sequence).
