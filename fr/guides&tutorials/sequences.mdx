Dans la section **“Séquences”**, vous pouvez créer et gérer le processus qui vous permettra d’envoyer, retarder et programmer des messages, ainsi que de gérer votre agent IA.

- Allez dans votre organisation en haut à gauche
- Cliquez sur “Sequences”
- Puis cliquez sur “Create Sequence”

<img style={{ borderRadius: "0.5rem" }} src="/image_s/1.png" />

La page “Create your sequence” va apparaître :

Sélectionnez votre support

<img style={{ borderRadius: "0.5rem" }} src="/image_s/2.png" />

## 📸 Instagram

<AccordionGroup>
  <Accordion title="Créer des séquences Instagram outbound">
  
    - Sélectionnez les déclencheurs entrants puis cliquez sur “+” et “add action step”.
  
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/1.png" />
  
    - Sélectionnez votre compte Instagram puis cliquez sur “continue”.
  
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/2.png" />
  
    Uniquement pour **"Story reply"** et **"Comment"**.

    - Créez votre message modèle puis cliquez sur "continue".

    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/3.png" />

    - Sélectionnez l’agent, le délai de réponse, puis cliquez sur "continue"

    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/4.png" />

  </Accordion>
  <Accordion title="Créer des séquences Instagram inbound">
  #### Pour le déclencheur "New Followers", vous suivez les mêmes étapes que pour inbound.
  - Sélectionnez les déclencheurs sortants puis cliquez sur “+” et “add action step”.
  - Pour "Followers (another account)", "New Likes (specific post)", et "Comments (another account)", ajoutez la source : utilisez @username pour les followers et l’URL du post pour les likes ou commentaires.
<img style={{ borderRadius: "0.5rem" }} src="/image_s_i_o/1.png" />

<img style={{ borderRadius: "0.5rem" }} src="/image_s_i_o/2.png" />

<img style={{ borderRadius: "0.5rem" }} src="/image_s_i_o/3.png" />
- Ajoutez une description de votre audience cible pour une efficacité optimale.
- Sélectionnez vos limites de DM de prospection
<img style={{ borderRadius: "0.5rem" }} src="/image_s_i_o/4.png" />
- Sélectionnez l’agent, le délai de réponse, puis cliquez sur "continue"
