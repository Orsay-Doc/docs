Dans la section **“Séquences”**, vous pouvez créer et gérer le processus qui vous permettra d’envoyer, retarder et programmer des messages, ainsi que de gérer votre agent IA.

- Allez dans votre organisation en haut à gauche
- Cliquez sur “Sequences”
- Puis cliquez sur “Create Sequence”

La page “Create your sequence” va apparaître :
Sélectionnez votre support

## 📸 Instagram

<AccordionGroup>
  <Accordion title="Créer des séquences Instagram Inbounds">
    - Sélectionnez les déclencheurs entrants, puis cliquez sur “+” et “add
    action step”.
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/1.png" />
    - Sélectionnez votre compte Instagram, puis cliquez sur “continue”.
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/2.png" />
    Uniquement pour **"Story reply"** et **"Comment"** :

    - **(Optionnel) Ajoutez des mots-clés déclencheurs** pour créer une **séquence spécifique**. La séquence ne se déclenchera que si le commentaire ou la réponse à la story contient l'un de ces mots-clés. Si aucun mot-clé n'est ajouté, c'est une **séquence globale** qui se déclenche sur tous les commentaires ou réponses aux stories. Les mots-clés sont insensibles à la casse et fonctionnent en logique OU.

    - **(Optionnel) Sélectionnez des publications ou stories ciblées** pour limiter la séquence à un contenu spécifique. Si aucune publication/story n'est sélectionnée, la séquence s'applique à toutes.

    <Tip>
    **Priorité de déclenchement :** Les séquences ciblant des publications/stories spécifiques sont vérifiées en premier, puis les séquences avec mots-clés, puis la séquence globale (sans mots-clés) en dernier recours.
    </Tip>

    - Créez votre message modèle, puis cliquez sur "continue".
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/3.png" />
    - Sélectionnez l’agent, le délai de réponse, puis cliquez sur "continue".
    <img style={{ borderRadius: "0.5rem" }} src="/image_s_i_i/4.png" />
  </Accordion>
  <Accordion title="Créer des séquences Instagram Outbounds">
    Configuration des Déclencheurs et Actions
	Commencez par sélectionner un déclencheur (trigger), tel que « New Followers » ou « New Like ».
	Ajoutez ensuite les actions qui s'exécuteront depuis votre compte Instagram. Toutes les actions ajoutées au même flow se déclencheront séquentiellement sur ce compte, si le déclencheur est activé.
	Exemple :

    Déclencheur : « New Like » (sur un post spécifique).
    Actions : « Follow » puis « Like ».
    Résultat : Si une personne like le post, le système suivra d'abord cet utilisateur, puis likera l'un de ses posts.

    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/2.1.png" /> - Ajoutez la
    source : utilisez @username pour les followers et l’URL du post pour les likes ou
    commentaires.
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/3.1.png" /> - Définissez
    le profil du prospect cible, puis cliquez sur "Continue".
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/4.1.png" /> - Cliquez sur
    le "+", puis sur "Add action step".
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/1.png" /> - Choisissez le
    type d'action que vous souhaitez effectuer , puis cliquez sur "Continue".
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/2.png" /> - Il est possible
    d'en accumuler plusieurs.Pour effectuer plusieur action avec la même sequence.
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/6.png" /> - Vous pouvez
    aussi ajouter un délai entre chaque action.
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/3.png" /> - Et en choisir
    la durée, que ce soit en heures ou en journée.
    <img style={{ borderRadius: "0.5rem" }} src="/image_istg/4.png" />
    <Tip>
    Nos recommendations: - 1.⁠ Follow - 2.⁠ Like un post - 3.⁠ Envoyer un message.
    </Tip>

  </Accordion>
</AccordionGroup>
