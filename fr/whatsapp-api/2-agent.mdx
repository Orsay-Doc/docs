L'**agent** prend le relais après le template approuvé par Meta. Même système d'agent que sur tous les autres canaux.

## Chemin rapide

<Steps>
  <Step title="Ouvrez Agents → + Créer un agent">
    Le plus rapide est **Guided AI Setup** — répondez à 8 questions et Orsay écrit le prompt pour vous.
  </Step>
  <Step title="Sauvegardez et publiez">
    Prévisualisez la conversation dans le panneau de test, puis **Publish**.
  </Step>
</Steps>

→ Guide complet : [Agents — guide complet](/fr/guides&tutorials/agents)

→ [Exemples de prompts & bonnes pratiques](/fr/resources/prompt)
