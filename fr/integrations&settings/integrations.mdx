Pour accéder à ce menu, cliquez sur **“Intégrations”** en bas à gauche.

Cliquez sur **“Parcourir les intégrations”**

<img style={{ borderRadius: "0.5rem" }} src="/image_integration/1.png" />

## Pour connecter “WhatsApp” :

<AccordionGroup>
  <Accordion title="Connecter WhatsApp">
    - Cliquez sur **“WhatsApp”**, puis **“Connecter WhatsApp”**
    - Ensuite sur **“Créer un profil WhatsApp”**
    - Remplissez vos informations, puis cliquez sur **“Confirmer et continuer”**
    - Cliquez sur **“Connecter le numéro WhatsApp”**
    - Une fois de retour sur [app.orsay.ai](http://app.orsay.ai/), cliquez sur **“Créer un message entrant”**
    - Entrez le message souhaité et cliquez sur **“Terminer et créer le lien”**

Votre profil WhatsApp est maintenant intégré avec succès.

## Importer des fichiers :

<AccordionGroup>
  <Accordion title="Importer">
    - Importez un fichier CSV
    - Remplissez le pays **“France”** (si français)
    - Prénom/Numéro de téléphone : renseignez le titre de la colonne correspondante
    - Sélectionnez la séquence appropriée
    - Cliquez sur **“Importer le fichier”**

Pour importer un CSV, vous devez avoir une séquence avec le type de déclencheur <code style={{ color: '#d00' }}>CONTACT_SUBSCRIBED_TO_SEQUENCE</code>
