## Etape 1 — Connecter votre compte WhatsApp

<Note>
**Deux facons de connecter WhatsApp :** Vous pouvez soit creer un nouveau compte WhatsApp Business, soit connecter votre compte existant.
</Note>

<AccordionGroup>
  <Accordion title="Option A : Creer un nouveau compte WhatsApp Business">
    1. Allez dans **Integrations** → **WhatsApp**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/1.0.png" />
    2. **Creez un nouveau profil** (ajoutez un nom + photo de profil)
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/2.0.png" />
    3. Cliquez sur **Connecter un nouveau numero WhatsApp**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/3.0.png" />
    4. Vous serez redirige vers Facebook — approuvez les premieres pages, puis selectionnez votre **Business Manager**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/4.0.png" />
    5. Creez un nouveau **compte WhatsApp Business**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/5.0.png" />
    6. Creez un nom au format : **"Nom - Votre Societe"**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/6.0.png" />
    7. Ajoutez un **numero de telephone** (non connecte a un autre WhatsApp — nous recommandons **On Off Business** pour obtenir facilement un numero)
    8. Validez le numero avec un **code 2FA**
  </Accordion>

  <Accordion title="Option B : Connecter votre WhatsApp Business existant">
    <Warning>
    Cette methode ne fonctionne que si vous utilisez **WhatsApp Business** (pas un compte personnel).
    </Warning>

    1. Allez dans **Integrations** → **WhatsApp**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/1.jpeg" />
    2. **Creez un nouveau profil** (ajoutez un nom + photo de profil)
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/2.jpeg" />
    3. Cliquez sur **"Connect WhatsApp number"**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/3.jpeg" />
    4. Vous serez redirige vers Facebook — approuvez les premieres pages, puis selectionnez votre **Business Manager**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/4.jpeg" />
    5. Selectionnez **"Connect with existing WhatsApp Business App"**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/5.jpeg" />
    6. Entrez votre numero de telephone
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/num/6.jpeg" />
    7. Scannez le QR code
    8. Validez avec un **code 2FA** et confirmez
  </Accordion>

  <Accordion title="Valider le numero de telephone">
    1. Allez sur **https://business.facebook.com/billing_hub**
    2. Selectionnez le business auquel vous venez d'ajouter un numero → allez sur **WhatsApp** et ajoutez un **moyen de paiement**
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/1.1.png" />
    3. Cliquez sur le bouton **"TEST"** pour valider votre numero — entrez votre WhatsApp personnel et cliquez sur **ENVOYER**. Si vous recevez un message, tout est pret.
    <img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_1/2.1.png" />
  </Accordion>
</AccordionGroup>

## Etape 4 — Creer une sequence

1. Allez dans **Sequences** et cliquez sur **Creer une sequence**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/1.0.png" />
2. Selectionnez **WhatsApp**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/2.0.png" />
3. Choisissez un type de declencheur :

| Declencheur | Quand il se declenche |
|---|---|
| **Message recu** | Chaque fois qu'un nouveau lead envoie un message a votre numero |
| **Contact cree** | Pour tout nouveau lead non encore assigne a une autre sequence |
| **Contact abonne a la sequence** | Pour les leads specifiquement assignes a cette sequence (utilise avec imports CSV & Zapier) |

4. Cliquez sur **"+"** pour ajouter une nouvelle etape
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/3.png" />
5. Selectionnez votre **compte WhatsApp**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/4.png" />
6. Selectionnez votre **message modele**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/5.png" />
7. Selectionnez l'**Agent**, definissez le **Delai de reponse**, ajoutez des **relances** si necessaire, puis cliquez sur **Confirmer**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/6.png" />
8. **Activez** la sequence et **publiez-la**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_3/7.png" />

## Etape 5 — Connecter Zapier a votre source de leads

Si vous obtenez des leads depuis des sources externes (Facebook Ads, landing pages, CRM, etc.), connectez Zapier pour que chaque nouveau lead arrive automatiquement dans votre sequence WhatsApp.

### Comment le configurer

1. Allez dans **Integrations** → cliquez sur **Connecter Zapier**
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/1.png" />

2. Dans Zapier, cliquez sur **+ Create** pour creer un nouveau Zap
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/2.png" />

3. Choisissez votre **source de leads** (Facebook Lead Ads, ClickFunnels, systeme.io, ou toute autre plateforme compatible)
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/3.png" />

4. Selectionnez **Create Lead for Sequence** comme action Orsay — cela ajoute chaque lead directement dans une sequence specifique
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/4.png" />

5. Mappez les champs : **Prenom**, **Nom**, **Numero de telephone**

6. Selectionnez l'**Organisation** et la **Sequence** correspondantes
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/6.png" />

<Tip>
Vous pouvez trouver l'ID d'une sequence en cliquant a cote du nom de la sequence dans la page Sequences.
<img style={{ borderRadius: "0.5rem" }} src="/image_zapier/5.png" />
</Tip>

Desormais, chaque lead capture depuis vos sources externes arrivera automatiquement dans Orsay et sera place dans la bonne sequence WhatsApp.

## Etape 6 (optionnel) — Envoyer une campagne en masse via CSV

Si vous avez une liste existante de leads, vous pouvez tous les importer d'un coup :

1. Allez dans **Integrations** → cliquez sur **Bulk Leads** → **Upload Files**
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_5/1.png" />
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_5/2.png" />

2. Importez votre fichier CSV :
   - Remplissez le **pays** ("France" si francais)
   - Mappez les colonnes **prenom** et **numero de telephone**
   - Selectionnez la bonne **sequence**
   - Cliquez sur **Upload File**

<Info>
Pour importer un CSV, vous devez avoir une sequence avec le type de declencheur `CONTACT_SUBSCRIBED_TO_SEQUENCE`.
</Info>

<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_5/3.png" />
<img style={{ borderRadius: "0.5rem" }} src="/image_whatsapp/step_5/4.png" />

3. Configurez le **calendrier automatique** :
   - Selectionnez la sequence d'upload
   - Definissez le nombre de leads a integrer par lot
   - Cliquez sur **Update Schedule**
